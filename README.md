## 주요 내용 및 기술 분석

### 1. 펄스 폭 변조 (PWM, Pulse Width Modulation)
- **개념**: 디지털 신호(5V, 0V)를 매우 빠른 속도로 전환하여, 평균 전압을 조절함으로써 마치 아날로그 전압이 출력되는 것처럼 만드는 기술
- **구현**: 아두이노에서는 `analogWrite(pin, value)` 함수(0~255)를 사용하며, 보드 상에 물결 기호(`~`)가 표시된 핀에서만 활성화됨.

### 2. 풀다운 저항 (Pull-down Resistor)
- **개념**: 입력 핀이 어디에도 연결되지 않아 주변 노이즈에 의해 0과 1을 불규칙하게 오가는 플로팅(Floating) 현상을 방지하기 위한 저항
- **원리 및 주의점**:
  - 버튼을 누르지 않았을 때, 입력 핀을 저항(주로 10kΩ)을 통해 GND와 연결함으로써 0V 상태를 유지
  - **쇼트(Short) 방지**: 만약 저항이 아닌 일반 전선으로 GND에 연결하면 버튼을 누르는 순간 5V와 GND가 다이렉트로 연결되는 쇼트(Short-Circuit)가 발생하여 장치가 고장날 수 있음.
  - **전류의 흐름**: 전류는 흐르기 쉬운 곳으로 가려는 성질이 있기 때문에 버튼이 눌릴 때 상대적으로 저항이 매우 높은 10kΩ 풀다운 저항 쪽이 아닌 아두이노 입력 핀 쪽으로 전류가 흘러 들어가게됨.

### 3. 가변저항(Potentiometer)
- **개념** : '5V' - '아날로그 입력핀' - 'GND'로 연결되어 있으며, 손잡이를 돌려 아날로그 입력핀에 들어가는 전압을 조절할 수 있음. 손잡이를 돌려 입력핀이 5V 단자에 밀착하면 5V 전압 전체가 걸려 `1023`을 읽고, GND 단자에 밀착하면 0V가 되어 `0`을 출력하는 방식.
- **궁금증** : 그렇다면 저항을 최소로 만들어 `1023`을 출력할 때(입력핀이 5V에 밀착할 때), 5V와 GND가 직접적으로 연결되어 쇼트(합선)가 일어날 수도 있는 것 아닌가?
- **해결** : 가변저항 내부의 5V 핀과 GND 핀 사이에는 손잡이 위치와 무관하게 항상 일정한 크기의 고정 저항(보통 10kΩ)이 끝까지 유지되는 구조임. 손잡이를 끝까지 돌려 저항이 0이 된다는 것은 입력 핀과 5V(또는 GND) 사이의 저항이 0이 되는 것일 뿐, 전체 전원 회로인 5V와 GND 사이에는 항상 큰 저항이 가로막고 있으므로 안전함.
